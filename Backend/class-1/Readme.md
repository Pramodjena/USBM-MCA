# NEM

## Node.js

- `npm init -y`
- In your package.json do the changes
- `"start": "node index.js"`

```js
{
  "name": "class-1",
  "version": "1.0.0",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1",
    "start": "node index.js"
  },
  "keywords": [],
  "author": "",
  "license": "ISC",
  "description": ""
}
```

- To run your backend app - `npm start`
